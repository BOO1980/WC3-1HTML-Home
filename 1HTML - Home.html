HTML Introduction

What is HTML?

HTML is the standard markup language for creating Web pages.

HTML stands for Hyper Text Markup Language

HTML is an embedded language

HTML describes the structure of Web pages using markup

HTML elements are the building block of HTML pages

HTML elements are represented by tags

HTML tags label pieces of content such as "heading", "paragraph", "table" and so on

Browser do not display the HTML tags, but use them to render the content of the page

HTML Tags

HTML Tags normally come in pairs like <p> and </p>

The first tag in a pair is the <b> start tag,</b> the second tag is the <b> end tag</b>

The end tag is written like the start tag, but with a <b>forward slash</b> inserted before the tag name

The browser uses the information inside the tags to decide how to treat the contents of the document

Tip: The start tag is also called the <b>opening tag, </b> and the end tag the <b>closing tag </b>

The <!DOCTYPE> Declaration

The <!DOCTYPRE> declaration represents the document type, and helps browser to display web pages correctly

It must only appear once, at the top of the page (before any HTML Tags).

The <!DOCTYPE> declaration is not case sensitive
